Conduit is a REST API-based web application for publishing articles and managing user accounts.

This project is focused on API testing in Postman and on organizing QA documentation in a clean portfolio structure. I built an independent Postman collection for the Conduit API, prepared test data in Pre-request scripts, and added automated checks for each request.

What I did
Worked with the Conduit API endpoints and organized requests by folders (Articles, Profile, Comments, and others)
Made requests independent by creating required data before execution (users, articles, comments)
Added collection-level variables (url, passwordConduit) and reused them across requests
Covered requests with tests (status codes and response content validation)
Reused repeating checks via helper functions at collection level
Exported the collection and added it to this repository
