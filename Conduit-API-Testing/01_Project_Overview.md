# Conduit – API & Web QA Project

## 📌 Project Overview

Conduit is a REST API-based web application used for publishing articles and managing user accounts.

This project focuses on API testing, requirement analysis, and structured QA documentation.

---

## 🎯 Scope of Work

During this project I performed:

- Requirements analysis (Registration page)
- Functional decomposition
- Test Plan creation
- API testing via Postman
- Test case design
- User Story validation

---

## 🔎 QA Activities

- Registration validation logic testing
- Input validation testing
- Negative testing scenarios
- API response validation
- Status code verification
- Error handling validation

---

## 🛠 Tools Used

- Postman
- GitHub
- Google Docs
- Markdown

---

## 📎 Key QA Focus Areas

- API contract validation
- Business logic validation
- Edge case testing
- Input boundary analysis
- Authentication & authorization checks
