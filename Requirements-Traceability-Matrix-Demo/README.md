This folder contains a demo of a Requirements Traceability Matrix (RTM) built from two sources: user personas and user stories.

The RTM shows how each requirement or user story is linked to detailed test cases. The spreadsheet also includes links to the original documents (user personas and user stories), which were used as the basis for creating the RTM.

What is inside
RTM demo spreadsheet (Google Sheets)
Links to user personas
Links to user stories
Test cases mapped to user stories (IDs like EC-001, EC-002, etc.)

How to review
Open the spreadsheet
Follow the links to user personas and user stories
Check how each user story is traced to test cases in the RTM
